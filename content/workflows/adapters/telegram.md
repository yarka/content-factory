# Telegram Adapter

Берёт готовый core draft (deaify-нутый) и форматирует под Telegram.

---

## Шаг 1: Адаптация длины и структуры

```
Адаптируй core draft под Telegram.

Правила формата:
- Длина: 200-1500 символов (если длиннее — сокращай, не обрезай смысл)
- Первая строка = заголовок, жирный: <b>Заголовок</b>
- Абзацы: 1-3 предложения, пустая строка между ними
- Финал: вопрос к аудитории или открытая мысль
- Тире: только короткие (–), никаких длинных (—)
- Хэштеги: только если указаны в `content/channel-dna.md`
- Ссылки: <a href="url">текст</a>

Если core draft длиннее 1500 символов:
- Оставь самое важное
- Убери повторения и вводные фразы
- Сохрани личную деталь и финальный вопрос

Верни ТОЛЬКО текст в HTML-формате. Без комментариев.
```

---

## Шаг 2: Финальная проверка

Перед выводом проверь:
- [ ] Первая строка жирная (`<b>...</b>`)
- [ ] Длина ≤ 1500 символов
- [ ] Все `—` заменены на `–`
- [ ] Ссылки в формате `<a href="...">текст</a>`
- [ ] Финал — вопрос или мысль, не призыв "подписывайтесь"

---

## Шаг 3: Вывод

```
─────────────────────────
TELEGRAM
─────────────────────────

{текст в HTML}

─────────────────────────
Символов: {число}
Рубрика: {из content/channel-dna.md}
─────────────────────────
```

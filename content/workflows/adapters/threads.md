# Threads Adapter

Берёт core draft и делает короткую версию для Threads.

## Входные данные

- Core draft: финальный текст после deaify
- Язык: русский (или EN если core draft был на EN)

## Особенности площадки

| Параметр | Значение |
|----------|----------|
| Язык | Русский (или EN если core был EN) |
| Длина | До 500 символов — жёсткий лимит |
| Формат | Plain text, короткие абзацы |
| Хэштеги | 1-2 максимум |
| Финал | Вопрос или открытая мысль |

**Принцип:** Один удар. Одна мысль. Ничего лишнего.

---

## Step 1: Find the Strongest Thesis

```
Прочитай core draft и найди одно из двух:

Вариант A — Сильнейший тезис:
- Самая неожиданная или контринтуитивная мысль
- То что заставляет остановиться и перечитать
- Конкретный вывод, а не общая идея

Вариант B — Тизер:
- Зацепка которая создаёт вопрос в голове
- Намёк на историю или инсайт без раскрытия
- Заставляет хотеть узнать продолжение

Выбери один вариант. Запиши выбранную мысль одним предложением.
```

---

## Step 2: Compress to 500 Characters

```
Напиши пост для Threads на основе выбранной мысли.

Жёсткие правила:
- Максимум 500 символов (считай включая пробелы и хэштеги)
- Голос автора из `content/channel-dna.md` — сохранить
- Никаких вводных слов: "Сегодня хочу поделиться..." = удалить
- Никаких объяснений зачем пишешь — сразу суть
- Короткие предложения предпочтительнее длинных

Структура (одна из):
- [Мысль] + [Почему это важно одним предложением] + [Вопрос]
- [Провокация] + [Короткое объяснение]
- [Личное наблюдение] + [Открытая мысль]

Финал: либо вопрос к читателю, либо незавершённая мысль которая остаётся в голове.
```

---

## Step 3: Telegram Link (опционально)

```
Если core draft — развёрнутый пост в Telegram-канале:

Добавь в конец (до хэштегов):
"Подробнее в канале → {ссылка на канал}"

Добавляй только если:
- Пользователь предоставил ссылку на канал
- Или ссылка есть в `content/channel-dna.md`

Если ссылки нет — пропусти этот шаг.
```

---

## Step 4: Final Check

Перед выводом проверь:

- [ ] Длина строго до 500 символов (считай сам)
- [ ] Хэштегов не больше 2
- [ ] Голос автора сохранён, не звучит как AI
- [ ] Нет вводных фраз-пустышек
- [ ] Есть финальная мысль или вопрос

---

## Step 5: Output

Выведи финальный пост в формате:

```
─────────────────────────
THREADS DRAFT
─────────────────────────

{текст поста}

─────────────────────────
Символов: {число} / 500
─────────────────────────

Готово к публикации. Правь или одобряй.
```

---

## Anti-Patterns

| Не делай | Почему |
|----------|--------|
| Пересказывать весь core draft | Threads = один удар, не дайджест |
| Больше 2 хэштегов | Выглядит как спам |
| Заканчивать на "что думаете?" без контекста | Слишком generic |
| Вводные абзацы с контекстом | Нет места — сразу суть |
| Сжимать всё подряд равномерно | Лучше выбросить половину, оставить лучшее |

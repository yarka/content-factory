## Концепция

Стиль привязан к адаптеру:
- **Telegram** → минимализм

## Референсы

**Что понравилось:**
- Lenny/Anthropic стиль: тёплый персиковый фон, жирный sans-serif хедлайн слева, иллюстрация/UI-элементы справа, sparkle-акценты, логотип внизу
- Это выглядит как HTML/CSS или Figma, не AI-генерация

**Баннер LinkedIn Макса:**
- Deep navy фон, нейросеть-нодов визуал, тёплый amber glow
- Потенциальная основа для тёмного варианта дизайн-системы

## Реализация — открытый вопрос

**Вариант А: HTML/CSS → Playwright → PNG**
- Пишем шаблон, подставляем заголовок из поста
- Текст всегда идеальный, цвета точные, дёшево
- Контроль 100%

**Вариант Б: Промпт → FLUX / DALL-E → PNG**
- Больше свободы для иллюстраций
- Текст на картинке часто глючит

**Вариант В: Гибрид**
- HTML для layout/текста + FLUX для декоративного фона/иллюстрации

**Рекомендация:** А или В. Макс упомянул "banana" как image API — уточнить что именно.

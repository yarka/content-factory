# Intelligence Manual Context

Use this file for optional enrichment that should influence discovery.

## Allowed inputs

- competitor hints
- analog accounts
- subreddit suggestions
- sites, blogs, newsletters, or channels to watch
- hypotheses about what language or hooks might matter
- audience pain notes from calls, DMs, or workshops

## Annotation format

For each note include:

1. type: evidence / hypothesis / preference
2. source or origin
3. why it matters
4. confidence

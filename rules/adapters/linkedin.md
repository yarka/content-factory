# LinkedIn Adapter

Берёт готовый core draft (deaify-нутый) и форматирует под LinkedIn.

## Входные данные

**Всегда читай перед запуском:**
1. `channel-dna-linkedin.md` — DNA специфичный для LinkedIn (цель, аудитория, голос, референсы)
2. `channel-dna.md` — основной голос автора
3. `rules/writing-guide.md` — стиль

Если `channel-dna-linkedin.md` не заполнен (пустые поля) → запусти онбординг:
> "LinkedIn DNA не настроен. Запустить? (это займёт 5-6 вопросов)"

## Особенности площадки

| Параметр | Значение |
|----------|----------|
| Язык | English only (см. channel-dna-linkedin.md) |
| Тон | 80% бизнес-инсайт, 20% личная история как контекст |
| Длина | **600–1200 символов**. Ёмко, не растягивать. |
| Формат | Plain text, никакого HTML |
| Переносы строк | Работают, пустые строки между абзацами обязательны |
| Хэштеги | **Нет** |
| Эмодзи | **Нет** (→ для списков допустимо) |
| CTA | Мягкий — вопрос или открытое наблюдение. Никогда "Like + Comment". |
| Тире | Только дефис (-) или em dash (—); не использовать en dash (–) |

**Критично:** Первые 2 строки — всё, что видно до кнопки "see more". Хук должен быть там.

---

## Язык

Прочитай поле "Язык" из `channel-dna-linkedin.md`:
- **EN** → переводи core draft на английский (Step 1 ниже)
- **RU** → не переводи, адаптируй стиль под LinkedIn
- **Оба** → создай две версии: сначала RU, потом EN
- **Не указан** → спроси: "На каком языке постим в LinkedIn — EN или RU?"

---

## Step 1: Translate / Adapt + Reformulate Hook

**Если язык = EN:**
```
Переведи core draft на английский.

Правила:
- Сохрани смысл и голос автора
- Первая строка = хук. Переформулируй под EN-аудиторию (founders, operators, AI builders)
- Убери детали которые не работают для EN-аудитории
- Личная история, которая создаёт доверие — оставь
- Не переводи дословно: адаптируй под то как звучат на LinkedIn по-английски

Хук — выбери один тип:
- Провокация: "Most companies are automating the wrong things."
- Число: "I saved 12 hours a week by changing one workflow."
- Противопоставление: "Everyone's talking about AI. Few are actually shipping with it."
- Личный момент: "Last month I almost hired someone for a role that didn't need to exist."
```

**Если язык = RU:**
```
Адаптируй core draft под LinkedIn на русском.

Правила:
- Сохрани голос автора из writing-guide.md
- Первая строка = хук (видна до "показать ещё")
- Тон: чуть профессиональнее чем в Telegram, без корпоративщины
- Аудитория: по channel-dna-linkedin.md

Хук — выбери один тип:
- Конфликт: "Все говорят про AI. Мало кто реально запустил."
- Число: "Я сэкономил 12 часов в неделю, поменяв один процесс."
- Личный момент: "На прошлой неделе я чуть не нанял человека на роль которой не существовало."
```

---

## Step 2: Structure Check

После перевода проверь структуру:

```
Структура LinkedIn-поста:
- Строка 1: хук (то что видно до "see more")
- Строка 2: пустая строка или продолжение хука
- Тело: 2-4 абзаца, каждый 1-3 предложения
- Финал: вывод или сдвиг перспективы
- CTA: вопрос к аудитории (последняя строка перед хэштегами)
- Хэштеги: 3-5 штук, отдельная строка

Пустые строки между абзацами — обязательны для читаемости.
```

---

## Step 3: Soft Close

```
Заверши пост мягко — один из вариантов:
- Открытый вопрос: "What's your take?" / "Curious what's working for you."
- Наблюдение без призыва: "That's the shift I'm seeing."
- Философский вывод: "The game changed. The question is when you'll play it differently."

НЕ добавляй хэштеги.
НЕ пиши "Comment X for access" или "Like if you agree".
```

---

## Step 4: Final Check

Перед выводом проверь:

- [ ] Первая строка — хук, работает без контекста
- [ ] Длина 600–1200 символов (не растягивать)
- [ ] Нет HTML-тегов
- [ ] Нет en dash (–) — только дефис (-) или em dash (—)
- [ ] Нет хэштегов
- [ ] Нет эмодзи (→ допустимо)
- [ ] CTA мягкий — вопрос или наблюдение, без "Comment/Like"
- [ ] Голос звучит человечески, не как корпоративный пресс-релиз

---

## Step 5: Output

Выведи финальный пост в формате:

```
─────────────────────────
LINKEDIN
─────────────────────────

{текст поста}

─────────────────────────
Characters: {число}
─────────────────────────
```

## Anti-Patterns

| Не делай | Почему |
|----------|--------|
| "I am excited to share..." | LinkedIn cliche №1 |
| Длинный вводный абзац без хука | Никто не доскроллит до "see more" |
| Хэштеги внутри текста | Ломает читаемость, не принято на EN LinkedIn |
| Перевод дословно | Звучит как Google Translate |
| Список из 10 пунктов | Теряет внимание, лучше 3-5 |
| Заканчивать на риторическом вопросе без CTA | Нет призыва к действию |

Раньше статья занимала полдня. Теперь — полчаса.

## Проблема

Писать статьи муторно. Не сам текст — а всё вокруг него:
- Искать источники
- Проверять факты
- Убирать "AI-звучание"
- Форматировать для блога
- Публиковать в Telegram

Каждый шаг отнимает время. Легко забыть про какой-то. Или сделать вполсилы.

## Идея

А что если агент сделает всё сам? Не отдельные куски — а весь pipeline от вопроса "о чём пишем?" до поста в канале.

## Как это работает

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Questions  │ ──▶ │  Research   │ ──▶ │   Write     │
└─────────────┘     └─────────────┘     └─────────────┘
      │                                       │
      │                                       ▼
      │             ┌─────────────┐     ┌─────────────┐
      │             │   Publish   │ ◀── │   Deaify    │
      │             └─────────────┘     └─────────────┘
      │                   │
      └───────────────────┘
           feedback loop
```

Пять фаз. Каждая делает одну вещь.

**Phase 1: Questions** — агент спрашивает про угол, аудиторию, главный takeaway. Без этого получается generic текст.

**Phase 2: Research** — [Exa](https://exa.ai) ищет источники. Агент собирает цифры, примеры, цитаты.

**Phase 3: Write** — черновик на 600+ слов. С ASCII-диаграммой. С инлайн-ссылками.

**Phase 4: Deaify** — четыре параллельных критика ищут AI-отпечатки. Шаблонные фразы. Ровный ритм. Абстрактные утверждения. Устаревшие версии софта.

**Phase 5: Publish** — сохраняет файл, коммитит, добавляет превью в очередь Telegram.

## Что изменилось

Раньше я откладывал статьи. Слишком много ручной работы.

Теперь пишу: "статья про X" — и через полчаса готово. Агент задал три вопроса, я ответил. Дальше он сам.

Качество не упало. Наоборот — деаификация находит штуки, которые я бы пропустил. "Важно понимать" — выкинуть. Три предложения одной длины подряд — разбить.

## Что дальше

Pipeline модульный. Можно добавить фазу review перед публикацией. Или заменить Telegram на Ghost/Notion. Или убрать research если пишешь из личного опыта.
